Gevalideerde eisen

In het project Portfolivo zitten verschillende stukken Javascript. Voor deze stukken heb ik redelijk wat onderzoek gedaan waarmee ik vooral stukken gebruikt heb die ik zelf nog aangepast heb zodat mijn website er meer professioneel uitziet en aan mijn wensen voldoet.

Portfolivo is dus een project waar ik steeds meer POC's in verwerkt heb.

Proof of Concepts

Alle Proof of Concepts die ik gemaakt heb voor Portfolivo.
Dit is voor mij de eerste keer dat ik Javascript gebruik in websites.

Scroll knop animatie

See the Pen jOWmJqo by Ivo Kruijssen (@ivo-kruijssen) on CodePen.

De animatie van deze knop heeft niks te maken met javascript maar is volledig in css gemaakt, het javascript gedeelte heeft te maken met het klikken op de knop. Als je op de knop klikt word je met een animatie verder gestuurd naar een aangegeven ID.

Je hebt jQuery nodig om er een mooie trendy animatie aan toe te voegen, waarmee je de duur van de animatie en soort animatie kunt aanpassen.

Animatie op basis van muisbeweging

See the Pen zYrwewQ by Ivo Kruijssen (@ivo-kruijssen) on CodePen.

In deze animatie haal ik met Javascript de positie van mijn muis op, en gebruik ik de TweenMax Library om de Y as van de afbeelding daarop aan te passen, zodat de afbeelding in een tegenovergestelde richting gaat dat hoe je de muis beweegt.

Zelf ben ik in de code gaan onderzoeken wat er allemaal gebeurd als ik bepaalde dingen weg haal en zo heb ik bijvoorbeeld gevonden dat ik de X as weg kon laten en alleen de Y as kon gebruiken wat er beter uit ziet.

Animatie op basis van scroll

See the Pen PoZmLXg by Ivo Kruijssen (@ivo-kruijssen) on CodePen.

Op de landingspage wilde ik een animatie maken met een delay op basis van omhoog/omlaag scrollen. Deze animatie wordt door meerdere Library's waar gemaakt.

Weer ben ik in deze code dingen gaan veranderen zodat ik het beter kon begrijpen. Ik heb het nu zo gemaakt dat als de animatie inlaad hij meteen 80 pixels omhoog schiet.

Slideshow

See the Pen JjGJoWB by Ivo Kruijssen (@ivo-kruijssen) on CodePen.

Dit is de POC waar ik het meeste van heb geleerd.

Voor de posters op de Design pagina was ik opzoek naar hoe ik een image slideshow kon maken. Toen ben ik op w3schools gaan kijken en had ik er een gevonden die ik aangepast had waardoor ik begrijp hoe zo'n slideshow werkt en dat ik hem naar mijn eigen wensen kan aanpassen.

Toen ik deze slideshow had aangepast probeerde ik hem te gebruiken voor 5 andere posters. Dit ging helemaal fout omdat ik dezelfde parameters gebruikte voor elke poster die eigenlijk alleen voor de eerste poster bedoeld waren.

Door dingen te testen ben ik hier uiteindelijk achtergekomen en heb ik extra parameters aangemaakt waardoor het wel werkt. :)
