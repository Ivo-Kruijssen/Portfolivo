Professionele toolset

Om er voor te zorgen dat mijn producten er professioneel uitzien heb ik gebruik gemaakt van verschillende applicaties uit het Adobe Creative Cloud pakket dat ik gekocht heb, zoals Photoshop, Adobe Xd en Premiere Pro.
Op mijn vorige opleiding heb ik met een gecrackte Photoshop gewerkt dus wist ik al een beetje hoe ik hier mee moest werken, maar ik heb tijdens deze periode heel veel geleerd.

De code-editor die ik gebruik is PHP-storm. Deze code-editor heb ik op mijn vorige opleiding ook gebruikt dus weet ik hoe ik er gemakkelijk mee om kan gaan. En ik gebruik een lokale WAMP server waar ik mijn website op test.

Visuele ontwerpen

Design producten die ik gemaakt heb zoals posters en banners en animaties.

Op de afbeelding hierboven zie je een banner die ik gemaakt heb voor op de pagina's van de leeruitkomsten. Deze afbeelding heb ik in Photoshop gemaakt en ik heb hiervoor de kleuren van het kleurenpallete gebruikt die ik gebruik voor deze site.

Hier zie je de bollen die ik op de landingspage boven de achtergrond laat bewegen op basis van de cursor. Dit heb ik in Photoshop gemaakt omdat ik dacht dat de landingspagina er nog redelijk leeg uitzag. De achtergrond van deze afbeelding is eigenlijk transparant zodat ik deze makkelijker kan laten bewegen met Javascript, maar ik heb er nu een achtergrond onder gemaakt zodat het makkelijker te zien is.

Dit is de animatie van de Ikai website die ik gemaakt heb voor Ikai. De auto komt in het scherm met een animatie zoals je hierboven ziet, en als je dan op een link klikt op het wegwijzer bord rijdt de auto het scherm uit. Deze animatie is heb ik volledig in CSS gemaakt.
